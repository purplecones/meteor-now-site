Instantly deploy Meteor apps with meteor-now for free!

Get Started With Three Simple Steps

Install Dependencies 

$ npm install -g now meteor-now

Create Free Account 

$ now --login

Deploy Meteor App 

$ meteor-now

meteor-now
is the closest thing to 
meteor deploy
that is available today. It uses a combination of open-source technology and amazing cloud hosting initiatives to provide a free option to host your Meteor apps.
